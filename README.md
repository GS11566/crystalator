# crystalator
A simple simulation of salt crystallization.
## How to use
Create a venv
```shell
python -m venv .venv
```
Activate it
```shell
# For example on Linux
source .venv/bin/activate
```
Install numpy and pygame
```shell
pip install numpy pygame
```
And run!
```shell
python crystallize_numpy.py
```
## I want to customize it!
The code has a bunch of variables at the top; read the variable names, change them however you want.
